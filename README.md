# Factura — 홈 화면 설치형 웹앱 (PWA)

앱스토어 없이 **아이폰·안드로이드 홈 화면에 추가**해서 전체화면 앱처럼 쓰는 버전입니다.
한 번 열어두면 **인터넷 없이도** 동작합니다(QR 생성기 내장 + 오프라인 캐시).

## 폴더 구성
```
factura-pwa/
  index.html            ← 앱 본체 (이 파일이 시작점)
  manifest.webmanifest  ← 앱 이름/아이콘/전체화면 설정
  sw.js                 ← 오프라인 캐시(서비스워커)
  icons/                ← 앱 아이콘(192/512/180)
  .nojekyll
```

## 1) 웹 주소로 올리기 (GitHub Pages, 무료)
아이폰은 로컬 파일을 홈에 추가할 수 없어서 **웹 주소가 필요**합니다. 가장 간단한 무료 방법:

1. GitHub에서 새 저장소(repository) 생성 (예: `factura`)
2. 이 `factura-pwa` 폴더 **안의 내용물**을 저장소 루트에 업로드(드래그&드롭 가능)
   - `index.html` 이 저장소 최상위에 오도록
3. 저장소 → **Settings → Pages** → Source를 `Deploy from a branch`, 브랜치 `main` / 폴더 `/ (root)` 선택 → Save
4. 1~2분 뒤 표시되는 주소(예: `https://<아이디>.github.io/factura/`)가 앱 주소입니다

> 꼭 GitHub Pages가 아니어도 됩니다. https로 정적 파일을 올릴 수 있는 곳이면(Netlify, Cloudflare Pages 등) 어디든 동일하게 동작합니다.

## 2) 아이폰(Safari)에서 홈 화면에 추가
1. **Safari**로 위 주소 열기 (Chrome 아님)
2. 하단 **공유 버튼**(□↑) → **홈 화면에 추가** → 추가
3. 홈 화면 아이콘으로 실행하면 주소창 없는 전체화면

## 3) 안드로이드(Chrome)에서 홈 화면에 추가
1. **Chrome**으로 위 주소 열기
2. 우측 상단 **⋮ 메뉴 → 홈 화면에 추가 / 앱 설치**
3. 설치하면 일반 앱처럼 전체화면 실행

## 사용
- 켜면 회사명 / Tax ID / 주소가 크게 표시 + 항목별 QR
- 항목을 **탭하면 풀스크린 단일 QR**(다른 코드와 섞이지 않게)
- 우측 상단 ⚙ → 정보 입력 / 표시 언어 선택(독·프·영·스·헝)
- 비워둔 언어는 영어(기본) 정보를 그대로 이어받아 표시

## 정보 수정 후 갱신
`index.html` 내용을 바꿔 다시 올렸는데 옛 화면이 보이면, `sw.js`의 `CACHE = 'factura-v1'`을
`'factura-v2'`처럼 올린 뒤 다시 올리면 새 버전이 적용됩니다.
